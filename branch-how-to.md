Пара строк о том, как пользоваться ветками:
=======
Как создать ветку
        $  git branch name

Переходим в ветку
        $ git checkout name

После коммита версии ветки пушим её на сервер
        $ git push

Но у ветки нужно указать происхождение, поэтому добавляем
        $ git push --set-upstream origin name
